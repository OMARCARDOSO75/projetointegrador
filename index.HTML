DO QUE SE TRATA ESTE TRABALHO

Este projeto explora a análise de dados através de utilização de dispositivos de Internet das Coisas (IoT) para o monitoramento de áreas protegidas e a prevenção de queimadas. Com base nesses dados, o projeto investiga como sistemas de alerta precoce podem ser implementados para detectar e responder rapidamente a potenciais focos de incêndio, ajudando a mitigar os danos ambientais e protegendo ecossistemas vulneráveis, através de sensores, esses equipamentos são instalados em áreas a serem preservadas formando um tipo de cerca virtual eletrônica, os dispositivos quando reconhece a mudança de temperatura, enviam seus dados para um gateway, onde suas informações são enviadas para nuvem, a plataforma envia um sinal de alerta para as autoridades locais, o sensor DHT foi utilizado para captura de dados de umidade e temperatura em um período aproximado de um ano para coleta e armazenamento dos dados que serão analisados ara este projeto, alguns sistemas como o FAEDO desenvolvido pela empresa Indra que trabalha com desenvolvimento de soluções e tecnológicas avançadas utiliza sensores térmicos com a capacidade de identificar a temperatura de uma região com cobertura de 20 quilômetros de distância, tem a capacidade através de um modelo 3D localizar detalhadamente sua localização para facilitar a chegada ao foco de incêndio.
